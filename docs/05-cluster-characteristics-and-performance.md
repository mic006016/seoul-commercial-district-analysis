**1. Cluster 0: 직장인 야간 회식 상권 (성장 과대예측 구간)**

- **상권 정체성:** 평일 매출(91.8%), 저녁 매출(44.2%), 3050세대 비중(80.7%)이 4개 군집 중 압도적 1위임. 주말(8.2%)과 심야(0.2%)는 완전히 죽어있는 전형적인 오피스 회식 상권임.
- **성능 지표:** Growth_Recall(성장형 재현율) 0.719 (최상) / Accuracy 0.548 (최하)
- **분석 결과:** 모델이 해당 군집에서 '성장' 신호를 매우 민감하고 공격적으로 잡아냄. 실제 성장하는 상권을 놓치지 않고 70% 이상 색출하지만, 오답(유지/위축을 성장으로 오판) 역시 빈번하게 발생해 전체 정확도를 깎아먹음. 3050 직장인이라는 단일 수요층의 미세한 매출 상승을 모델이 과대 해석하는 경향이 있음.

**2. Cluster 1: 주야간 균형 복합 상권 (모델 최적화 구간)**

- **상권 정체성:** 주말 매출 34.1%, 점심(27.4%)-오후(28.7%)-저녁(31.6%) 비중이 매우 고르게 분포함. 3050(63.4%) 중심에 2030(29.7%)이 섞인 안정적인 배후 주거+업무 복합 상권임.
- **성능 지표:** 샘플 수 최다(6099개) / Accuracy 0.685 (최상) / Macro_F1 0.596 (최상)
- **분석 결과:** 본 예측 모델의 '스윗 스팟(Sweet Spot)'. 특정 시간대나 연령에 치우치지 않아 데이터의 노이즈가 적음. 시계열 모멘텀(QoQ) 등 모델이 학습한 규칙이 가장 안정적이고 정확하게 맞아떨어지는 모범 군집임.

**3. Cluster 2: 2030 심야 유흥 상권 (모델 사각지대)**

- **상권 정체성:** 심야 매출(32.0%)과 2030세대 비중(41.2%)이 전 군집 최고치임. 유행 변화가 극심하고 변동성이 큰 대학가 및 야간 유흥 상권임.
- **성능 지표:** Accuracy 0.684 (높음) / Growth_Recall 0.464 (최악)
- **분석 결과:** 치명적인 한계점이 노출된 구간임. 높은 Accuracy는 단지 대부분을 '유지형(1)'으로 보수적으로 찍어서 얻어걸린 착시 현상임. 실제 '성장형' 상권은 절반도 맞추지 못함. 2030 중심 심야 상권 특유의 비선형적인 트렌드 급변과 단기 변동성을 현재의 거시적 매출 피처만으로는 전혀 설명하지 못하고 있음.

**4. Cluster 3: 주간 업무 중심 상권 (저성장 안정 구간)**

- **상권 정체성:** 점심 매출(34.0%) 1위, 저녁 매출(17.7%) 꼴찌. 평일(70.8%)에만 수요가 발생하고 퇴근 후에는 텅 비는 공단, 관공서 인근 등 주간 특화 상권임.
- **성능 지표:** Accuracy 0.606 / Growth_Recall 0.605
- **분석 결과:** 점심 수요라는 확실한 앵커(Anchor)가 존재해 예측 난이도는 무난함. 성장을 예측하는 재현율도 60%대로 방어했으나, 저녁 소비가 부재하여 Cluster 1만큼의 폭발적인 성장이나 신뢰도 높은 예측력을 보여주지는 못함.

### [결론 및 모델 한계점]

- **예측력의 양극화:** 본 모델은 소비 패턴이 정형화된 3050 평일 중심 상권(Cluster 1, 0)에서는 우수한 변별력을 발휘함.
- **비정형 데이터의 한계:** 심야/유흥/2030 상권(Cluster 2)에 대한 성장 타겟팅 기능이 붕괴된 상태임.
- **개선 방향:** 유흥 상권의 성장을 선제적으로 포착하기 위해서는 현재의 매출 기반 지표를 넘어, SNS 언급량 급증, 신규 프랜차이즈 진입률, 단기 공실률 변화 등 '유행의 속도'를 측정할 수 있는 외부 대체 데이터(Alternative Data) 수혈이 필수적임.
